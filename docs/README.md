## ⚡ Quick Start

1. Install dependencies: `pnpm install`
2. Configure `.env` file
3. Initialize database: `npm run db:init`
4. Start bot: `npm run dev`

Full details: [Quick Start Guide](setup/QUICKSTART.md)
